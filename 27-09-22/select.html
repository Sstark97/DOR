Select

El elemento (<select>) de HTML representa una selección de opciones. Las opciones contenidas en el menú son representadas por elementos <option>, los cuales pueden ser agrupados por elementos <optgroup>

Ejemplo

<select name="select">
    <optionvalue ="value1"> Value 1 </option>
    <optionvalue ="value2"> Value 2 </option>
    <optionvalue ="value3"> Value 3 </option>
</select>
