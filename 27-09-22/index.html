TextArea

Área de Texto multilínea sin formato

Button

Representa un botón que el usuario puede pulsar para desencadenar una acción

FieldSet

Nos permite organizar en grupos los campos dentro de un formulario

Legend

Es la etiqueta que nos permite introducir datos dentro de un formulario

DataList

Agrupa varios elementos <option>, una alternativa al <select>

Output

Nos sirve para mostrar el resultado de un calculo, o el resultado de la acción de un usuario

Option

Es la representación de un item dentro de la etiqueta <select>

OptGroup

Agrupa una serie de <option> dentro de una etiqueta <select>
